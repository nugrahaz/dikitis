Required Permissions

To provide a better experience, this app uses connection to get all the information shown in this app, collect usage data for providing and improving the services, and does not sell any user's information to any third party. These are the permissions the app only uses to show meaningful information to the user.

We want to make sure that you fully understand the permission we need to make Dikitis app function how you want. It's vital that you are clear on how we use your data. You can choose to provide a few optional permission (such as notification, storage, and camera) depending on the features you intend to use.

Compulsory Permissions

These permissions are required for the application's most of the features and functions. Not granting them will break some of the features albeit the application will continue to work.

  • READ_MEDIA_IMAGES

    • Used to read image files to add a photo profile from storage.

  • CAMERA

    • Used to take a picture to add a photo profile via the camera

  • NOTIFICATION

    • Used to post notifications regarding disease diagnosis results.

Automatic Permissions

These permissions are granted automatically by the system.

  • INTERNET

    • Allows applications to use the network or internet.

  • ACCESS_NETWORK_STATE

    • Used to check if network is available and is connected.

  • READ_EXTERNAL_STORAGE

    • Used to read image files from storages to add a photo profile from storage. Due restricted starting in Android 13 (Tiramisu), this permission has no effect, apps need permission READ_MEDIA_IMAGES.

If you do not agree to use the services mentioned earlier, then you may not be able to use some of the features of this application. However, you can still use the basic features.
